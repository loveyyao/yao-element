### 公共年份范围选择器 yao-date-picker

> 公共年份范围选择器。年份格式`yyyy`。

# 属性

| 参数          | 说明      | 类型                | 可选值            | 默认值     |
|-------------|---------|-------------------|----------------|---------|
| value       | `value` | `String`,`Number` |                | `[]`    |
| clearable   | 清空      | `Boolean`         | `true`、`false` | `false` |
| disabled    | 禁用      | `Boolean`         | `true`、`false` | `false` |
| placeholder | 提示文本    | `String`          |                |         |

# 事件

- `change`：值发生变化是触发，参数`value`
- `show`：选择器打开
- `close`：选择器关闭
